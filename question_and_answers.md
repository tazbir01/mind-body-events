###### 1. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let greeting;
greetign = {};
console.log(greetign);
```

- A: `{}`
- B: `ReferenceError: greetign is not defined`
- C: `undefined`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: A: {};

<i>When you assign an empty object to a variable, the variable holds a reference to that object. So, when you use console.log(greeting), it logs the content of the greeting variable, which is an empty object {}.</i>

</p>
</details>

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {
  return `Hi there, ${name}`;
}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: B: Hi there, undefined

<i>The name variable is not declared or initialized before it is used. The name variable is only declared inside of the sayHi() function. This means that it does not exist outside of the function. If you call the sayHi() function without passing an argument, the name variable will be undefined.</i>

</p>
</details>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C: 3

<i>
The answer is 3 because the forEach() method iterates over all of the elements of the nums array and executes the callback function for each element. The callback function checks if the current element is non-zero. If it is, the function increments the count variable.</i>

</p>
</details>
